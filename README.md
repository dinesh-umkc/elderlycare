FastAPI is used for rest services
- Step 1 - Run these commands to install fast api and uvicorn

        pip install fastapi
        pip install uvicorn
        pip install sentence-transformers

- Step 2 - Go to the project folder in command prompt
- Step 3 - execute

        uvicorn main:app --reload
